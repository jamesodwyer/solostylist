**Submission:**
1. Compute `starts_at` and `ends_at`:
   ```typescript
   const startsAt = `${date}T${slotTime}:00.000Z`  // Note: UTC. This is acceptable for MVP.
   const totalMinutes = selectedServices.reduce((sum, s) => sum + s.duration_minutes, 0)
   const startDate = new Date(startsAt)
   const endDate = new Date(startDate.getTime() + totalMinutes * 60000)
   const endsAt = endDate.toISOString()
   ```
2. Call `createAppointment`:
   ```typescript
   startTransition(async () => {
     const result = await createAppointment({
       client_id: selectedClient.id,
       starts_at: startsAt,
       ends_at: endsAt,
       notes: notes || undefined,
       override_working_hours: overrideHours,
       services: selectedServices.map(s => ({
         service_id: s.id,
         service_name: s.name,
         service_price: s.price,
         service_duration_minutes: s.duration_minutes,
       })),
     })

     if (result.warning && !overrideHours) {
       setWarning(result.warning)
       return  // Show warning, don't close
     }
     if (result.error) {
       setError(typeof result.error === 'string' ? result.error : 'Failed to create appointment')
       return
     }
     // Success
     onOpenChange(false)
     router.refresh()
   })
   ```
3. If warning received and user hasn't overridden yet: display warning, user can check override and re-submit
4. If error: display in red text below the button
5. If success: close sheet, `router.refresh()` triggers diary re-fetch
