The Money page is a Server Component that:
- Authenticates via `getUser()` (redirect to /login if no user)
- Reads `searchParams.date` (or defaults to today: `new Date().toISOString().split('T')[0]`)
- Queries payments for the selected day:
  ```typescript
  const dayStart = `${dateStr}T00:00:00.000Z`
  const dayEnd = `${dateStr}T23:59:59.999Z`
  const { data: payments } = await supabase
    .from('payments')
    .select(`
      id, amount, method, payment_type, notes, paid_at,
      reference_payment_id,
      clients(first_name, last_name),
      appointments(starts_at)
    `)
    .eq('owner_user_id', user.id)
    .gte('paid_at', dayStart)
    .lte('paid_at', dayEnd)
    .order('paid_at', { ascending: false })
  ```
- Computes daily totals in the Server Component (integer pennies, no floats):
  - `totalCash`: sum of amounts where `payment_type === 'payment' && method === 'cash'`
  - `totalCard`: sum of amounts where `payment_type === 'payment' && method === 'card'`
  - `totalRefunds`: sum of amounts where `payment_type === 'refund' || payment_type === 'void'`
  - `grossTotal`: totalCash + totalCard
  - `netTotal`: grossTotal - totalRefunds
