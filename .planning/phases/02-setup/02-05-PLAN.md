- Navigate to Settings tab (bottom nav)
- Verify business profile info is displayed (trading name, working hours summary, slot size)
- Tap "Services" link → should go to /settings/services

- Set deposit: Fixed £10.00, Required
- Save → verify service appears in list under "Hair" category
- Verify "Deposit" badge is visible
- Tap service to edit → change price → save
- Swipe left on service → verify it moves to "Hidden" section
- Swipe on hidden service → verify it reactivates
- Add an uncategorised service → verify it appears at top

**4. Client List (CLNT-01, CLNT-02, CLNT-03):**
- Navigate to Clients tab (bottom nav)
- Verify empty state message
- Tap "+" to add client: first name "Jane", last name "Smith", phone "07700 900000", enable marketing consent
- Save → verify client appears in list with initials "JS"
- Add 2 more clients with different first letters
- Verify A-Z section headers appear
- Search "jan" → verify Jane Smith appears
- Search "077" → verify phone search works
- Clear search → full list returns

- Remove a tag via x button → verify removal
- **Notes tab:** Verify empty state. Tap + → select "General" → write a note → save. Tap + → select "Treatment" → write a note → save. Verify both appear in timeline with type badges in reverse-chrono order. Edit a note → verify update. Delete a note → verify removal.
