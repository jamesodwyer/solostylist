<task type="auto">
  <name>Task 1: Create CSV export Route Handler</name>
  <files>src/app/api/export/[entity]/route.ts</files>
  <action>
Create a Next.js Route Handler at `src/app/api/export/[entity]/route.ts` that handles GET requests for 4 entity types: `clients`, `appointments`, `payments`, `notes`.

Authentication: Use `createClient()` from `@/lib/supabase/server` + `getUser()`. If no user, return `NextResponse.redirect(new URL('/login', request.url))`.

Entity routing via `await params` to get `entity` string. Return 404 Response for unknown entities.

Build a shared `buildCsv(headers, rows)` utility inline in the file:
- `escape(v)`: if value contains comma, double-quote, newline, or carriage return, wrap in double quotes and double-up any embedded quotes (RFC 4180)
- Prepend UTF-8 BOM (`\uFEFF`) for Excel Windows compatibility
- Use `\r\n` line endings (Excel-friendly)

Entity queries (ALL must include `.eq('owner_user_id', user.id)` as defence-in-depth even though RLS enforces it):

1. **clients**: Select `id, first_name, last_name, phone, email, address, marketing_consent, created_at`. Order by `created_at asc`. Headers: same field names.

2. **appointments**: Select `id, client_id, clients(first_name, last_name), starts_at, ends_at, status, notes, created_at, appointment_services(service_name, service_price)`. Order by `starts_at asc`. Flatten rows to include `client_name` (concatenated first+last) and `services` (pipe-separated service names like `"Haircut | Blow Dry"`). Headers: `id, client_id, client_name, starts_at, ends_at, status, notes, services, created_at`.

3. **payments**: Select `id, client_id, appointment_id, amount, method, payment_type, reference_payment_id, notes, paid_at, created_at`. Order by `paid_at asc`. Map `amount` from integer pennies to `amount_gbp` decimal string using `(amount / 100).toFixed(2)`. Headers: `id, client_id, appointment_id, amount_gbp, method, payment_type, reference_payment_id, notes, paid_at, created_at`.

4. **notes**: Query BOTH `client_notes` and `colour_formulas`, merge into a single array. For `client_notes`: select `id, client_id, note_type, content, created_at, updated_at`. For `colour_formulas`: select `id, client_id, formula, notes, created_at, updated_at` and map to same shape with `note_type: 'colour_formula'`, `content: formula + (notes ? '\n' + notes : '')`. Combine, sort by `created_at asc`. Headers: `id, client_id, note_type, content, created_at, updated_at`.

Response: Return `new Response(csv, { headers: { 'Content-Type': 'text/csv; charset=utf-8', 'Content-Disposition': 'attachment; filename="solostylist-{entity}-{YYYY-MM-DD}.csv"' } })`.

Do NOT use any npm CSV library. Do NOT use a Server Action for file download.
