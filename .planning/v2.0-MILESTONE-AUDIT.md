### Integration Gap Detail

**WeekView `onSlotPress` dead prop** (minor)

- `WeekView.tsx` declares `onSlotPress` in its props interface (line 25) and destructures it (line 50)
- Slot rows in the 7-day grid are bare `View` elements — no `TouchableOpacity` wrapper, no `onPress` handler
- DiaryScreen correctly passes `onSlotPress={(date, time) => handleSlotPress(time, date)}`
- **Impact:** Tapping an empty slot in Week view does nothing. Booking still works via:
  - FAB button (always available)
  - DayView slot taps (fully functional)
  - Month view day tap → switches to Day view → slot taps work
- **Fix:** Wrap slot `View` in `TouchableOpacity` with `onPress={() => onSlotPress(dateStr, slotTime)}`
