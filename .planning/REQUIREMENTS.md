# Requirements: v2.0 Native App

## Phase 7: Diary Calendar

- DIARY-01: Month view with dot indicators showing days with bookings
- DIARY-02: Week view showing 7-day overview with appointment blocks
- DIARY-03: Day view with time slot grid and positioned appointment blocks
- DIARY-04: Segmented control to switch between Month/Week/Day views
- DIARY-05: Date navigation (arrows, swipe, tap-to-select from month)
- DIARY-06: Booking flow — select client, select services, confirm with time/date
- DIARY-07: Appointment detail view — view, reschedule, mark complete/no-show/cancel
- DIARY-08: Services CRUD — create, edit, delete services with name/price/duration/category
- DIARY-09: Status-based colour coding (booked=blue, completed=green, cancelled=grey, no-show=amber)
- DIARY-10: Take Payment integration from appointment completion
- DIARY-11: Working hours enforcement with soft override
- DIARY-12: Double-booking prevention (database EXCLUDE constraint)
- DIARY-13: Client notes and colour formula visible during booking
- DIARY-14: Current time indicator (red line) on day view
- DIARY-15: Auto-scroll to current time (today) or working hours start (other days)
